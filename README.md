# Share My Favs
## 概要
自分の好きなアーティストを5組選ぶことでオリジナルの画像を作成でき、Twitterでシェアすることができるサービスです。
## URL
[Share my favs](https://share-my-favs.herokuapp.com/)
## 使用技術

- Ruby 2.6.6
- Ruby on Rails 6.0.3
- PostgreSQL
- JavaScript
- Tailwind CSS
- RSpec
- Git, GitHub
- Rubocop
- SpotifyAPI(wrapperのrspotify gemを使用)
- Heroku

## 機能一覧
- アーティストの検索(APIを使用)
- アーティストの追加(Ajax)
- アーティストの削除(Ajax)
- 画像の作成(minimagick)
- Twitterへのシェア
- OGPの作成
